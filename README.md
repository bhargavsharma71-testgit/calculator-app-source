# calculator-app-source-

Flask calculator app with Bootstrap UI for Azure DevOps CI/CD deployment

## Introduction
Simple calculator web app built using Flask.

## Getting Started
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## Features
- Add
- Subtract
- Multiply
- Divide

## Tech Stack
- Python
- Flask
- HTML/CSS (Bootstrap)
