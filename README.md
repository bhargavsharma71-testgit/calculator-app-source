# 🚀 DevOps Calculator App

A simple Flask-based calculator deployed using Azure DevOps CI/CD pipeline.

## 🔧 Features
- Add, Subtract, Multiply, Divide
- Clean UI
- Error handling
- CI/CD enabled

## 🛠 Tech Stack
- Python (Flask)
- Azure Web App
- Azure DevOps Pipelines

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
